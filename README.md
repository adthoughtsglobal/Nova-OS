## Introducing Nova OS - An OS Made For You.
![image](https://github.com/adthoughtsglobal/Nova-OS/assets/123654929/35409b32-4db2-4e40-af7f-878e612c235a)

Nova is a web based operating system that has an App Ecosystem powered a community of Creators like you.

# Includes
1. Operating system written in HTML, JS and CSS. (JSON is included in JS. Some libraries have been used.)
2. Default Apps for Nova OS package: Nova OS AppData BETA 

# A Friendly OS that's always with you.
Nova OS do not send and recive data from any external server to process your personal data in the safe hands. Your Data never escapes your device, all cache and personal data are saved in your Browser Storage. Making it an entirely local entity.
